# Newton-Raphson Method in $n$-dimensions

This is a repository containing supplementary material, figures and code relating to the pairs project for AMA3020 Investigations (Queen's University Belfast). 

The project investigates the generalized Newton-Raphson method in $n$-dimensions and its applications to mathematical optimization problems.

Files:
- `utils.py` : Helper functions for algorithms, plotting etc.

- `newton.py` : Collection of algorithms used in the project.

- `newton.ipynb` : Examples showcasing variations of Newton's method.

- `descent.ipynb` : Examples showcasing the Steepest Descent algorithm.

- `basins.ipynb` : Examples investigating basins of attraction.

- `comparison.ipynb` : Performance comparison of various methods.

- `derivations.pdf` : Derivations of one-dimensionsal and two-dimensional Newton-Raphson methods.
